![Casa Damiao](public/images/main-image.png?raw=true "Casa Damião")

## Getting Started

Launch database 

```bash
docker-compose up
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
